Concurrents

Cette partie va permettre de lister certains concurrents de Gun.

  • OrbitDB

    OrbitDB est une base de donnée Web décentralisée utilisant le Peer2Peer. Il utilise IPFS pour stocker des donnes et IPFS Pubsub pour synchroniser automatiquement les données des peers alors que Gun utilise les websockets

    Pour ce qui est de la résolution de conflits, OrbitDB utilise CRDTs, comme Gun

  • Ditto

    Ditto est comme les autres une base de donnée décentralisée utilisant le Peer2Peer. La différence de Ditto est que pour synchroniser les changements pour les peers, il est capable d'utiliser différents réseaux comme le Bluetooth, le WIFI, ...

    Pour ce qui est de la résolution de conflits, OrbitDB utilise CRDTs, comme Gun
